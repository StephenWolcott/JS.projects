NET Pay Program

Simple NET pay calculator. It takes your GROSS pay and then deducts Federal, State, FICA. It has the option to deduct any additional amount from your check. It then displays your NET pay.

  1. START
  2. DISPLAY "What is your GROSS pay per year?"
  3. INPUT GROSS pay
  4. FOR FEDERAL = GROSS * .15
  5. FOR STATE = GROSS * .05
  6. FOR FICA = GROSS * .0762
  7. DECLARE GROSS - (FEDERAL + STATE + FICA) = NET1
  8. DISPLAY "Do you have any additional withholdings?"
  9. INPUT ADDITIONALWITHHOLDING
  10. DECLARE NET1 - AdditionalWithholding = NET2
  11. DISPLAY "NET2 is your NET pay!"
  12. END

PTO v1.0

A simple program that calculates the PTO(Paid time off) for 1yr at a fixed rate of 5hrs per paycheck paid bi-weekly.

  1. START
  2. SET INT hrsPerPaycheck
  3. SET INT paychecksPerYear
  4. SET INT totalHrsPerYear = hrsPerPaycheck * paychecksPerYear
  5. DISPLAY "you get" totalHrsPerYear "hours of paid time off for 1 year"
  6. END

PTO v.2.0

A simple program that calculates the PTO(Paid time off) for the corresponding amount of time the employee has work in years. Returns amount of PTO in weeks.

  1. START
  2. SET INT HrsPerPaycheck
  3. SET INT PaychecksPerYear
  4. DISPLAY "how many years have you worked?
  5. INPUT YearsWorked
  6. if YearsWorked less than 5 HrsPerPaycheck = 5
  7. if YearsWorked equal to or greater than 5 HrsPerPaycheck = 6
  8. if YearsWorked equal to or greater than 10 HrsPerPaycheck = 7
  9. SET INT TotalHrsPerYear = HrsPerPaycheck * PaychecksPerYear
  10. SET REAL weeks = TotalHrsPerYear/40
  11. DISPLAY "you get" weeks " weeks off a year"
  12. END

  1. START

  9. SET INT TotalHrsPerYear = HrsPerPaycheck * PaychecksPerYear
  10. SET REAL weeks = TotalHrsPerYear/40
